# 📈 Quantitative Stock Dashboard

A live stock dashboard built with Python, Streamlit, SQL, and Alpha Vantage API.

## Features
- Fetch real stock data for any ticker
- View price chart, daily returns, and volatility
- Compare two stocks normalized to 100
- Sharpe ratio and key statistics
- Data stored in SQLite database

## Built with
- Python, Streamlit, Pandas, SQLite, Alpha Vantage API
